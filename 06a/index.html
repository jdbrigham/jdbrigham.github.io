Task: mute volume in certain sections of a video

ffmpeg -i /VIDEO-PATH -af "volume=enable='between(t,273,278)':volume=0","volume=enable='between(t,346,352)':volume=0","volume=enable='between(t,449,457)':volume=0","volume=enable='between(t,775,778)':volume=0","volume=enable='between(t,933,940)':volume=0","volume=enable='between(t,1072,1075)':volume=0" -c:v copy /NEW-VIDEO-PATH

Notes:

  • the (t,449,457) parts of the command tell where to start and where to stop muting (in seconds)
  • I’ve not been able to make this work in AppleScript b/c of the use of double quotes within the command itself.
  • A command to QuickTime in AppleScript (set current_time to current time as integer) can be used to determine a time code in seconds from the open video [see script “QT_currentTime.scpt”]

Task: export a series of frames from a video

ffmpeg -i /VIDEO-PATH -ss 00:02:49 -r 29.97 -s 1920x1080 -q:v 2 -vframes 230 /SAVE-PATH/frame%03d.png
