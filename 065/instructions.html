🗺️ Route Animator - Prep Instructions

Requires local server to test local files

  1. On iPhone, in Google Maps go to
    1. Profile photo
    2. Settings
    3. Location & Privacy
    4. Export timeline data
  2. Open the full timeline in JSON editor
  3. Use date range filters to edit to a single month
  4. Export filtered results and rename to month-year.json
  5. On iPhone, in the LeapSecond app, click on the month name and save the video
  6. Use Handbrake to create a video named month-year.mp4
  7. Duplicate a month-year.html file and edit lines 270, 271 and 273
  8. Add all new files to directory
